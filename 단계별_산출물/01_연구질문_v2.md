## 1. 연구 질문 (RQ) — v2

| # | RQ | 가설 | 핵심 방법 |
|--|---|:-:|---|
| RQ1 | 9~24세 (연령군 1~5)의 AI·디지털 의존 양상은 단계별로 어떻게 다른가? | H1 | 단계별 기술통계, 가중치 적용 |
| RQ2 | AI 사용 빈도·시간은 자기조절·수면·학업 시간과 어떤 통계적 연관(방향·크기)을 갖는가? | H1 | 다중회귀, β·CI, FDR 보정 |
| RQ3 | AI 사용 빈도는 **창의적 표현 활동(SNS 창작·여가창작·진로 자기효능)** 과 어떤 통계적 연관을 갖는가? | H2(축소) | 다중회귀, 효과크기 |
| RQ4 | 잠재계층분석으로 도출되는 사용자 군집은 **외부변수(학업성취·정신건강·또래관계)** 로 어떻게 차별화되는가? | H1·H2 | LCA → 외부변수 ANOVA |
| RQ5a | 미디어 이용시간의 코로나 전후(2018~2024) 단절시계열 변화는? | H1 | ITS 분석, KYRBS 가중치 |
| RQ5b | AI 챗봇 사용 빈도(2023~2024 횡단)는 단계별로 어떻게 다른가? | H1 | NIA 횡단, χ² + Cramer's V |

## 2. 부가설 (SH) — v2

| # | 부가설 | 효과 방향 | 검정 | 출처 |
|--|---|:-:|---|---|
| SH1 | AI 사용 빈도는 후기청소년·성인진입기에서 가장 높다. | 단조 증가 | ANOVA + 사후검정 | NIA |
| SH2 | 일평균 미디어시간 ≥ 4h 그룹은 < 4h 그룹보다 자기조절 점수가 낮다. | – | t·회귀 | KYRBS·KCYPS |
| SH3 | AI 사용 빈도와 평일 학업·수면 시간 사이에 음의 상관이 있다. | – | Pearson r·회귀 | NIA·KYRBS |
| SH4 | 디지털 콘텐츠 생산자(SNS 게시·블로그·유튜브)는 비생산자보다 진로 자기효능감·자기결정도가 높다. | + | 로지스틱·회귀 | KISDI·KCYPS |
| SH5 | AI 사용 빈도와 **창의적 표현 활동** 빈도는 양의 상관을 갖는다. | + | 회귀 | NIA·KISDI·여가부 |
| SH6 | 과의존 위험군 안에서도 창의적 표현 활동이 활발한 하위군은 학업·수면 영향이 약하다. (**조절효과**) | 조절 | Hayes PROCESS | NIA·KYRBS |
| SH7a | 2020년 코로나 이후 미디어 이용시간이 단절적으로 증가한다. | + | ITS | KYRBS |
| SH7b | AI 챗봇 사용 빈도는 후기 단계로 갈수록 높고, 가구 SES와 양의 연관이 있다. | + | 다중회귀 | NIA·KCYPS |
| SH8 | 가구 SES(부모 학력·소득)는 AI 사용의 **소비형 vs 생산형** 패턴을 차별화한다. | 패턴 차이 | LCA + ANOVA | KCYPS·통계청 |

## 3. 방법론 — v2

### 3.1 분석 게이트 (사전 점검)
1. **변수 가용성 게이트**: KCYPS·KYRBS·KISDI·NIA에서 RQ별 핵심 변수 보유 여부 확인. 미보유 시 변수 대체 또는 SH 삭제.
2. **표본 게이트**: 단계×성별 셀 30명 미만이면 인접 단계 결합.
3. **다중비교 게이트**: 검정 수 산출 후 FDR(Benjamini-Hochberg) 또는 Holm 보정 적용.
4. **사전등록**: OSF 또는 자체 git tag 로 가설·분석계획 사전 동결 (변경 시 이력 기록).

### 3.2 통계 기법
- **기술통계**: 가중치 적용, 95% CI 보고
- **추론통계**: 다중회귀(연속), 로지스틱(이분), 다항(범주). 효과크기 우선
- **종단**: KCYPS 패널 고정효과 모형 (가용 시)
- **단절시계열**: ITS — 분절 회귀(segmented regression), 자기상관 처리(Newey-West SE)
- **잠재계층분석**: 1~6 군집 비교, BIC + 엔트로피 + 3-fold CV + 외부변수 검증
- **매개·조절**: Hayes PROCESS 모델 4(매개)·1(조절)
- **민감도**: 결측 다중대치(MICE) m=5, 가중치 vs 비가중치 비교

## 6. 차별성 명시 (Critic 결함 ⑧ 반영)

같은 한국 공공데이터를 사용한 선행연구가 다수일 가능성 인지. 본 연구의 차별성:

1. **9~24세 5단계 분기** — 청소년기본법 정의 준수, 일반 통념(13~18) 초월
2. **H1·H2 동시 검증 설계** — 의존성·창의적 표현을 같은 모델 안에서 양립 가능성으로 조사

## 7. 한계·바이어스 — v2 (Critic 8 결함 통합)

## 8. Planner v2 자기비판 — 가장 큰 약점 3가지 (재선정)

### ① **H2를 다운그레이드해도 여전히 약하다**
"창의적 표현 활동" 도 결국 자기보고·빈도 변수이며, 창의성 본질에 닿지 못한다. 본 논문의 H2 결과는 **"AI 사용과 자기표현 빈도의 통계적 연관" 까지** 가 한계. 이걸 정직하게 4종 출판물 모두에 명시해야 함. → Writer 단계 톤 가이드.

### ② **마이크로데이터 신청 일정이 논문 일정을 좌우한다**
KCYPS·KYRBS·KISDI 마이크로데이터 신청·승인은 보통 1~3개월. 사용자가 즉시 신청해도 **1차 분석 가능 시점은 1개월+**. 그 사이에는 공개 통계·집계 자료로 1차 윤곽만 가능. → 논문 일정 2단계로 나눌 것 (윤곽 → 정밀).

### ③ **GitHub 공개 = 외부 비판 자동 수신**
4종 동시 공개 + GitHub 공개 = 통계학자·청소년정책 전문가의 비판이 빠르게 들어올 수 있음. 약점·한계를 미리 솔직히 적어두지 않으면 신뢰 손상. → Writer 단계 톤 가이드 + 한계 섹션 의무.
