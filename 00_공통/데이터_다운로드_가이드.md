## 3. 마이크로데이터 — 신청 절차 필수 (4종)

> 모두 무료. 회원가입·이용약정서·연구계획서 필요. 평균 1~3주 승인.

### 3.1 KCYPS (한국아동·청소년패널) — NYPI

- **신청처**: https://www.nypi.re.kr/archive
- **회원가입** → 데이터 신청서 작성 → 이용약정서 체결
- **요구사항**: 연구목적·기간·산출물 계획
- **저장**: `_신청대기/KCYPS/` (도착 후)
- **연락**: kcyps2018@nypi.re.kr

### 3.2 KYRBS 원시자료 — 질병관리청

- **신청처**: https://www.kdca.go.kr/yhs
- 원시자료 신청 → 연구목적 심사 → 이용약정 → 데이터 다운로드
- **승인 기간**: 약 1~4주
- **저장**: `_신청대기/KYRBS/`

### 3.3 KISDI 미디어이용행태 마이크로데이터 — KISDI

- **신청처**: https://data.kisdi.re.kr (또는 KISDI 도서관)
- 회원가입 → 자료신청서 → 이용약정 → 다운로드
- **저장**: `_신청대기/KISDI_미디어/`

### 3.4 MDIS 통계청 마이크로데이터

- **신청처**: https://mdis.kostat.go.kr
- 회원가입 (비교적 빠름) → 과제 신청 → 다운로드
- **권장 자료**: 인구주택총조사·사회조사·청년사회경제실태조사
- **저장**: `_신청대기/MDIS/`

## 5. 도착 후 처리

자료가 도착하면:
1. 해당 기관 폴더에 저장 (파일명 규칙: `연도_자료명_타입.확장자`)
2. `00_공통/데이터/원본/INDEX.md` 갱신 (✅ 표시 + 메타)
3. `analysis_scaffold.py` 의 `--mode=microdata` 호출 가능
4. 결과를 4종 .qmd 의 v2에 반영
