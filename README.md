## 두 가지 가설

- **H1**: 청소년들은 AI에 의존성이 심하다.
- **H2**: 청소년은 AI와의 대화로 창의성을 더 발휘할 수 있다.

본 연구는 두 가설을 **양립 가능** 으로 재정식화하고, 한국 공공데이터로 1차 검증한다.

---

## 1차 핵심 발견 (2025)

- 청소년 **67.9%** 가 생성형 AI 사용 (한국청소년정책연구원 2024)
- 청소년 스마트폰 과의존 위험군 **42.6%** (NIA 2024, 9년 연속 상승)
- **2025년 AI 디지털교과서** 76종 정규 도입 (초3·4·중1·고1)
- 청소년이 부모·친구보다 **AI 챗봇에 고민을 먼저** 토로하는 새 양상
- 미국 13~17세 26%(Pew 2025) → **한국이 약 2.6배 빠른 확산**

→ 같은 기간 의존과 창의적 표현이 **동시 증가**. H1·H2 양립 가능 가설 1차 시사.

## 7-에이전트 워크플로우

본 연구는 디시인사이드 thesingularity 갤러리 1153110번 글에 소개된 다중 에이전트 연구 자동화 시스템을 **GPT 패밀리 + 논문 작성용** 으로 재구성한 7-에이전트 워크플로우로 진행됐다.

```
[1] Planner    ─ 연구 질문·가설·방법론 설계
[2] Sentinel   ─ 윤리·청소년 개인정보·법규 검수
[3] Critic     ─ 가설·방법론·통계 결함 비판
[4] Reviewer   ─ 한국·해외 선행연구 조사·문헌리뷰
[5] Analyst    ─ 공공데이터 수집·정제·통계 분석
[6] Writer × 4 ─ ① 학술 ② 정책 ③ 백서 ④ 블로그 각각
[7] Reporter   ─ 초록·요약·발표용·인덱스
```

각 에이전트의 시스템 프롬프트와 산출물:
- [`에이전트_프롬프트/`](에이전트_프롬프트/) — GPT CLI 복붙용
- [`단계별_산출물/`](단계별_산출물/) — 각 에이전트 1차 산출물

운영·결정 로그:
- [`_운영로그.md`](_운영로그.md) — Supervisor (시각·산출·문제·개선)
- [`_의사결정로그.md`](_의사결정로그.md) — 모든 선택의 기록
